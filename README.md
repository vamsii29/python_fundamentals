
# 🐍 Python Fundamentals

> *Documenting my learning journey through Python — one exercise at a time.*

Based on **"Python Crash Course"** by Eric Matthes.

---

## 📌 Purpose

The goal of this repository is to build a strong foundation in Python through **active practice**, not passive reading.

I focus specifically on the **"Try It Yourself"** exercises provided in the book — reinforcing each concept by writing real code.

---

## 🗂️ Structure

```
python_fundamentals/
│
└── try_yourself/
    └── day1/
        ├── <exercise_files>.py
```

| Folder / File | Description |
|---|---|
| `python_fundamentals/` | Root repository folder |
| `try_yourself/` | All practice exercises from the book |
| `day1/`, `day2/`, ... | Exercises grouped by day |
| `.py files` | Individual solutions for each problem |

---

## 🧠 Approach

- ✅ Only **"Try It Yourself"** exercises are implemented in code
- 📅 Each day has its own folder to track progress over time
- 🔍 Code is written with a focus on **clarity and understanding**, not optimization

---

## 📝 Notes

This repository is intended for:

- 💻 Building a visible coding habit
- 📚 Creating a structured record of progress
