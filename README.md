Python Fundamentals

This repository documents my learning journey through Python fundamentals, based on the book “Python Crash Course” by Eric Matthes.

Purpose

The goal of this repository is to build a strong foundation in Python by actively practicing concepts rather than passively reading.
I am focusing specifically on the “Try It Yourself” exercises provided in the book to reinforce understanding through hands-on coding.

Structure

The repository is organized in a simple and scalable way:

python_fundamentals/
│
└── try_yourself/
    └── day1/
        ├── <exercise_files>.py

	•	python_fundamentals/ → Root repository folder
	•	try_yourself/ → Contains all practice exercises from the book
	•	day1/ → Exercises completed on Day 1
	•	.py files → Individual solutions for each “Try It Yourself” problem

Approach
	•	Only “Try It Yourself” exercises are implemented in code
	•	Each day has its own folder to track progress over time
	•	Code is written with a focus on clarity and understanding, not optimization

Progress Tracking
	•	Day 0: Started With hello.py

More days and exercises will be added as I continue progressing through the book.

Notes

This repository is intended for:
	•	Personal learning and consistency
	•	Building a visible coding habit
	•	Creating a structured record of progress
